# Решение задач из курса весеннего практикума для студентов 1-го курса 1 потока ВМК МГУ
**Язык программирования: NASM**

Привет! Этот репозиторий создан для того, чтобы помочь вам сориентироваться в весеннем практикуме. Здесь собрана основная информация о том, как устроен курс и как не потерять баллы

## Структура курса:
*Получение условий и сдача кода проходит на платформе Ejudge.*

- 4 домашних контестов из 6 небольших заданий, направленные на реализацию алгоритмов или отработку навыков, полученных на семинаре

- 2 больших проектных домашних задания, направленных на:
  1) Сравнение методов сортировки
  2) Вычисление корней уравнений и определенных интегралов

К реализации проектных заданий необходимо подготовить развернутый отчет по выполненной работе. Подробное описание каждого проектного задания находится в README.md внутри директорий с реализацией

*Важно учитывать: варианты заданий для каждого студента индивидуальные, посылки проверяют на антиплагиат*

## Система оценивания:
- Каждое верное решение задачи, защищенное перед проверяющим, дает **1 балл** вне зависимости от количества предыдущих посылок с неверным решением
- Сдача задачи после дедлайна постепенно снижает количество получаемых баллов вплоть до **0**. Старайтесь закрывать контесты вовремя!
  
## Структура репозитория:

Задания из курса разбиты по папкам согласно следующему шаблону:
```text
    ├── taskN
    │ ├── taskN_1.asm
    │ │   ...
    │ ├── taskN_M.asm
    │ ├── taskN_R1.c
    │ │   ...
    │ └── taskN_RM.c
    │
    ├── taskN_big
    | ├── reportN
    | | ├── reportN.pdf
    │ | └── reportN.tex
    │ ├── <realization_file1>.c
    │ ├── <realization_file2>.asm
    │ │   ...
    │ ├── Makefile
    │ └── README.md

```
где N - номер контеста, M - номер задачи в контесте N
