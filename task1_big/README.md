# Домашнее задание №1. Методы сортировки

**Выполнил** : Огай Владислав (103 гр.)

## Постановка задачи
Реализовать два метода сортировки  массива  чисел  и  провести  их  экспериментальное  сравнение.

## Вариант задания
1. **Тип элемента:** `Вещественные числа двойной точности (double).`.
2. **Тип сортировки:** по неубыванию модулей.
3. **Методы сортировки:** Метод «пузырька» + Быстрая сортировка, рекурсивная реализация. 

## Реализация
Программа представляет собой единственный файл `sort.c`

----------------------

Опции компиляции, используемые для сборки решения, находятся в файле `gcc-options.txt`.
Можно скопировать их в настройки собственной IDE, либо использовать этот файл при компиляции из командной строки:

```console
gcc sort.c @gcc-options.txt
```
